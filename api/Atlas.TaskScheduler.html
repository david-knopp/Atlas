    Class TaskScheduler

    Handles scheduling and distributing task execution between frames, attempting to balance frame-by-frame execution so one frame isn't heavier than the another.

    AddTask(IScheduledTask, UInt32)

    Registers the given task for scheduled updates, with an automatic phase assignment. This method will be a bit slower than AddTask(IScheduledTask, UInt32, Int32), as it will evaluate the number of scheduled tasks for upcoming frames in an attempt to select an optimum phase offset for the given task.
